## streamlit 활용

---

- 파이썬의 Streamlit
    - 웹 어플리케이션 제작 가능
    - 쉽고 간단한 API 제공으로 낮은 진입 장벽
- Streamlit UI
    - 다양한 UI 구성 요소 활용 가능
    - 텍스트, 데이터프레임, 위젯, 레이아웃 등

### Streamlit

- Data Science를 위한 맞춤형 웹 어플리케이션을 만들 수 있는 파이썬의 라이브러리
- 데이터 분석 결과를 손쉽게 공유할 수 있는 라이브러리
- 간결하고 명확한 API를 제공하며, 다른 프레임워크와 비교하여 상대적으로 진입장벽이 낮음
