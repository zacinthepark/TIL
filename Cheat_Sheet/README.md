## CLI (GIT BASH)

- 위아래 화살표 누르기
    - 이전에 내렸던 명령 히스토리들이 뜸
- code .
    - 현재 위치에서 vscode 열기
- python del.py
    - del.py 파일 실행
    - python은 ~~~~pyhton.exe에 대한 환경변수
- CLI에서 파일 수정
    - vi 파일명
    - i → 수정 → esc → :wq

## VSCODE

- 새로운 파일 생성
    - 왼쪽 위 아이콘 클릭
    - ctrl + n → ctrl + s
- 전체선택
    - ctrl + a
- 주석처리
    - ctrl + /
- 전체 주석처리
    - ctrl + a → ctrl + /
- 커서를 위아래 여러 줄로 확장
    - alt + ctrl + 위아래 화살표
- 줄 바꿈
    - alt + 위아래 화살표
- 줄 복사
    - alt + shift + 위아래 화살표
- 특정 단어 replace
    - ctrl + d
- 파일 settings
    - ctrl + ,
- 터미널 열기
    - ctrl + shift + `
- 시작화면
    - ctrl + k → f
